A browser-based chessboard project built with TypeScript.

## Overview

This project aims to build a clean and maintainable chessboard application for the browser, starting with board rendering and piece placement, and gradually adding movement rules, turn handling, captures, and full chess logic.

The main goal is to practice TypeScript, DOM rendering, state management, and application structure through a real project.

## Current Scope

At this stage, the project is focused on:

- Rendering an 8x8 chessboard
- Displaying the initial piece setup
- Building a scalable foundation
- Preparing the base for legal move validation

## Tech Stack

- TypeScript
- Vite
- HTML
- CSS

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Alextc35/browser-chess-ts
cd browser-chess-ts
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```
